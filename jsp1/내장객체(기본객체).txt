JSP에서 제공되는 내장(기본)객체 : 스크립트 영역에서 사용됨
						   생성,선언 필요없이 모든 jsp 에서 사용이 가능.

8+1(내장객체 종류)
1. ★request★  객체    : 요청객체. 브라우저에서 요청 한 정보를 저장. 
2. ★response★ 객체    : 응답객체. 브라우저로 명령전달 객체.
3. ★pageContext★객체  : 페이지별로 자원과흐름을 관리 하는 객체.(페이지하나하나 관리하는 객체)
4. ★session★  객체	   : 브라우저 상태 정보를 저장하는 객체 
5. ★application★ 객체 : 웹어플리케이션 정보를 저장하는 객체.
6. out 객체	         : 브라우저로 정보를 전달하기위한 출력스트림.
7. page 객체		     : jsp 페이지 자체.
8. comfig 객체        : 서블릿에 파라미터값을 전달하기위한 객체 

9. exception 객체	     : 예외객체. 예외페이지에서만 제공되는 객체
